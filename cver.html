cver.version()

CVER | Versioning


A very simple and minimal script for automated version counting. It can create and initialise a VERSION file and can increase its value each time it runs.


Usage

Command Line Tool


cver doesn't take any arguments.


Examples:

A VERSION file will be created inside the current directory:


$ python3 cver.py

Functions

cver.version ( int sub_max , int rev_max , int build_max , int main=0 , int sub=1 , int rev=0 , int build=0 )


cver can also be used a module. It provides a single function cver.version which can be used to create, initialise the VERSION file and increase its value.

Note: The sub_max, rev_max and build_max arguments are also responsible for the leading zeros in front of the version numbers. So for example, if one wants to limit the build values of the version number to 999 — so any time it became 1000, the build value will be 0 and the rev value will be increased by 1 — the notation will always be ddd because the length of the input (999) is 3.


Return: void
