CENV | Environment Generator


Sets up a minimal C environment. It is very practical to test ideas and short code snippets quickly or to start a new project from scratch without writing all the boilerplate. The generated makefile is very extensible and maintainable.


Dependencies

Python 3


This module is written in Python 3, therefore it needs the interpreter to be installed on your system to run it. I suggest you to use a package-manager to keep your installations ordered, clean and upto-date.


Macintosh: one can use homebrew:


brew install python3

Windows: one can use chocolatey:


cinst python3

Linux: on Ubuntu and LinuxMint it is already installed:


sudo apt-get install python3

Usage

Command Line Tool


cenv is command line tool, and takes two arguments. The first is the path to the directory where the project will be generated. The second one is the file name of the main C file.


Examples:

Create application.c and makefile in the project folder. Then compile file and run it, to check if everything went fine :


$ python3 cenv.py project application
$ cd project
$ make
$ ./build/application
